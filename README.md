# Temporally Similar Structure-Aware Spatiotemporal Fusion of Satellite Images
This is a demo code of the method proposed in the following reference:

R.Isono and S. Ono
``Temporally Similar Structure-Aware Spatiotemporal Fusion of Satellite Images.''

Update history:
Mar 6, 2026: v1.0 

For more information, see the following

- Published paper: https://ieeexplore.ieee.org/document/11404183

# How to use
Run ```main.m```
